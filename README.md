OKitty is a lightweight and extensible open source game engine built on top of SDL3. It is designed for developers who want a simple but powerful foundation for creating multi-platform 2D games, 3D games, tools, and general purpose applications. The engine focuses on clarity, modularity, and developer control while staying small and dependency friendly.

OKitty is licensed under the Apache License 2.0 which allows commercial and non-commercial use, modification, and distribution.

# [üß©] Features
OKitty avoids overblown abstraction and instead provides a clean and direct environment for real time applications. Some core design goals include:
* Multi-platform support using SDL3 as the backend
* A straightforward window and rendering system suitable for both 2D and 3D workflows
* A cross-platform input layer that unifies keyboard, mouse, and controller handling
* A math and utility layer designed around practical game development needs
* Modular architecture that allows you to use only the components you need
* Designed for beginners and advanced users who prefer direct access to rendering behavior
* Easily integrates with custom rendering backends, shaders, physics engines, and scripting systems
